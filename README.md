# Inbridge Nigeria Ltd - Static Website

This folder contains the fully static version of the Inbridge Nigeria Ltd website.

## Structure

- `index.html`: Home page
- `about/index.html`: About page
- `services/index.html`: Services overview
- `services/epc/index.html`: EPC Services
- `services/energy/index.html`: Energy Solutions
- `services/minerals/index.html`: Mineral Exports
- `contact/index.html`: Contact page
- `js/main.js`: JavaScript for interactions (mobile menu, animations)
- `attached_assets/`: Images and logos

## How to Run

Simply open `index.html` in any web browser. No server or build step is required.

## Dependencies

- **Tailwind CSS**: Loaded via CDN (`https://cdn.tailwindcss.com`)
- **Lucide Icons**: Loaded via CDN (`https://unpkg.com/lucide@latest`)
- **Fonts**: Inter (Google Fonts) and System Fonts

## Notes

- All links are relative.
- Images use relative paths.
- The site is fully responsive.
